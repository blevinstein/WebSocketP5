WebSocketP5
===========

A WebSocketServer for Processing, based on webbit.

**!! Looking for a download ??**  

Note, that this is the development project of the library!

To download a ready to use build, please go to: http://muthesius.github.io/WebSocketP5/ and pick the latest version (which is [0.1.4](http://muthesius.github.io/WebSocketP5/websocketP5-0.1.4/) currently).


Have fun!
